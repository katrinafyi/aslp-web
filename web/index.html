Online partial evaluation #

Online partial evaluation runs the simplification of the ARM specifications every time the semantics are requested. This is described in the Lift-off paper. 


          

        

        

          
Offline partial evaluation #

          

            Offline partial evaluation performs the simplification once, ahead-of-time, and then translated to Javascript.
            When requesting semantics, the offline lifter is simply evaluated as-is.
            This is described in the Lift-offline paper.
